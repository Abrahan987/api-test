Documentación Rápida

Cómo usar la API

1. Obtén tu API Key (en la sección Dashboard)
2. Agrega el header a tus requests: X-API-Key: tu_clave
3. Haz requests a los endpoints
4. Recibe imágenes aleatorias cada vez

Rate Limit: 100 requests cada 15 minutos
Formato: JPEG/PNG

Documentación Completa

Autenticación

Todos los endpoints requieren una API Key válida en el header X-API-Key.

Ejemplo:
X-API-Key: sk_user_abc123def456...

Respuestas

200 - OK: Imagen devuelta correctamente
401 - No autorizado: API Key inválida o faltante
429 - Límite excedido: Has sobrepasado el rate limit
500 - Error interno: Contacta con soporte
